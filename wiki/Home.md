# RentEase Server Wiki

Welcome to the **RentEase Server** documentation.

RentEase is a modern property rental platform built with focus on reliability, scalability, and maintainability. This backend service manages properties, users, bookings, and payments.

## 🚀 Overview

- **Language:** TypeScript
- **Framework:** Express.js
- **Database:** PostgreSQL with Prisma ORM
- **Architecture:** Clean Architecture with SOLID principles
- **Dependency Injection:** `tsyringe`

## 📚 Key Sections

- [[Architecture]]: Deep dive into the OOP and SOLID principles used.
- [[Authentication]]: Comprehensive guide to the registration, login, and OTP flows.
- [[Workflows]]: Understand how a request flows through the different layers.

## 🛠 Tech Stack

- **Core:** Node.js, TypeScript, Express
- **Database:** PostgreSQL, Prisma
- **Auth:** JWT, Bcrypt, OTP (Email/Phone), Google OAuth
- **Validation:** Zod (presumably, let me check)
- **DI:** tsyringe
